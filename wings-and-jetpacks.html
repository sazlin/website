Wings and Jetpacks: My new blog

Date Sun 29 June 2014

Redbull may give you wings, but Python gives you Jetpacks.

This entire site was created using Python and other common web tech. Here's the breakdown: This site uses Pelican, a static content generator written in Python. I'm personally writing my posts using MarkDown, but I also mix in some Liquid via the "Liquid" Pelican plugin. A mobile-first frontend is achieved with minimal effort by using bootstrap, which is easily integrated with Pelican via pelican-bootstrap3. The styling itself is a modestly customized version of the spacelab theme that is found on BootsWatch The entire site is hosted on github.io, though some older content still lives on Tumblr Fabric is used for deployment The site's repro lives on GitHub I edit everything in Sublime Text 2 on a MacBook Pro

I'm still configuring the site to my liking, so I'll leave this post as-is for now. I'll share more about what I do with the site later once I've locked down a look and feel that I like.
